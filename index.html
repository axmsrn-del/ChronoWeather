Политика конфиденциальности Privacy Policy

Последнее обновление: май 2026 · @ChronoWeather_bot Last updated: May 2026 · @ChronoWeather_bot
01

Общие положения

Настоящая Политика конфиденциальности описывает, какие данные собирает бот @ChronoWeather_bot («Бот»), как они используются и как защищаются.

Используя Бот, вы соглашаетесь с условиями данной Политики. Если вы не согласны — пожалуйста, прекратите использование Бота.

General

This Privacy Policy describes what data @ChronoWeather_bot ("the Bot") collects, how it is used, and how it is protected.

By using the Bot, you agree to this Policy. If you do not agree, please discontinue use of the Bot.

02

Какие данные собираются

Бот собирает только минимально необходимый набор данных:

  • Telegram User ID — числовой идентификатор для отправки уведомлений по подпискам
  • Telegram Username — только при отправке обращения в поддержку, для идентификации отправителя
  • Выбранные маршруты и подписки — для персонализированных уведомлений об изменении прогноза
  • Текст обращений в поддержку — если вы самостоятельно отправили сообщение через функцию обратной связи
✦ Бот не запрашивает и не хранит: имя, фамилию, номер телефона, геолокацию, фото, платёжные данные или любую другую персональную информацию.

Data collected

The Bot collects only the minimum data required to function:

  • Telegram User ID — a numeric identifier used to deliver subscription notifications
  • Telegram Username — only when you submit a support request, to identify the sender
  • Selected routes and subscriptions — to send personalised forecast change alerts
  • Support message text — only if you voluntarily send a message via the feedback feature
✦ The Bot does not request or store: full name, phone number, location, photos, payment details, or any other personal information.
03

Как используются данные

  • Отправка push-уведомлений при изменении прогноза погоды по подписанным маршрутам
  • Ответ на обращения в поддержку
  • Восстановление подписок при перезапуске сервиса

Данные не используются в рекламных целях, не передаются третьим лицам и не продаются.

How data is used

  • Sending push notifications when the weather forecast changes for subscribed routes
  • Responding to support requests
  • Restoring subscriptions after service restarts

Data is not used for advertising, is not shared with third parties, and is not sold.

04

Хранение данных

Данные хранятся в локальной базе данных на сервере Бота. Доступ к базе данных ограничен и защищён.

Данные хранятся до тех пор, пока активна подписка. При отписке от всех маршрутов ваши данные могут быть удалены по запросу.

Бот не использует облачные хранилища, аналитические сервисы или трекеры.

Data storage

Data is stored in a local database on the Bot's server. Access to the database is restricted and secured.

Data is retained while subscriptions are active. Upon unsubscribing from all routes, your data may be deleted upon request.

The Bot does not use cloud storage, analytics services, or trackers.

05

Сторонние сервисы

Для работы Бот взаимодействует со следующими внешними сервисами:

  • Telegram Bot API — платформа для доставки сообщений
  • Google Weather API (WeatherNext 2) — источник метеоданных. Запросы содержат только координаты маршрутных точек, без каких-либо пользовательских данных
  • Google Maps — опциональное открытие маршрута по ссылке в браузере пользователя

Запросы к Google API не содержат и не передают данные пользователей Бота.

Third-party services

The Bot interacts with the following external services:

  • Telegram Bot API — message delivery platform
  • Google Weather API (WeatherNext 2) — weather data source. Requests contain only route point coordinates — no user data
  • Google Maps — optional route link that opens in the user's browser

API requests to Google do not include or transmit any Bot user data.

06

Ваши права

Вы вправе в любой момент:

  • Отписаться от всех маршрутов — кнопка «Отписаться» в меню Бота
  • Запросить удаление всех своих данных — через обратную связь в Боте
  • Запросить информацию о том, какие данные о вас хранятся

Запросы на удаление данных обрабатываются в течение 7 рабочих дней.

Your rights

You may at any time:

  • Unsubscribe from all routes — use the "Unsubscribe" button in the Bot menu
  • Request deletion of all your data — via the Bot's feedback feature
  • Request information about what data is stored about you

Data deletion requests are processed within 7 business days.

07

Дети

Бот не предназначен для лиц младше 13 лет. Мы сознательно не собираем данные детей. Если вам стало известно, что ребёнок использует Бот — пожалуйста, свяжитесь с нами.

Children

The Bot is not intended for users under the age of 13. We do not knowingly collect data from children. If you become aware that a child is using the Bot, please contact us.

08

Изменения политики

Мы можем обновлять данную Политику. Дата последнего обновления указана в заголовке документа. При существенных изменениях подписчики Бота будут уведомлены.

Changes to this Policy

We may update this Policy from time to time. The date of the latest revision is shown in the document header. In the event of material changes, Bot subscribers will be notified.

Вопросы по конфиденциальности — через обратную связь в Боте. Privacy questions — via the feedback feature inside the Bot.
